# Cloud Bet Blitz Roadmap

## Vision

Cloud Bet Blitz is a modern online betting platform designed with a secure, scalable, and modular architecture.

The long-term goal is to support:

- Aviator
- Sports Betting
- Casino Games
- M-Pesa Deposits & Withdrawals
- Referral System
- Bonuses
- VIP Levels
- Admin Dashboard
- Analytics
- Real-time Multiplayer Games
